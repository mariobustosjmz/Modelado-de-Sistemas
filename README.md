#Modelado de Sistemas

>Simulaciones de Sistemas usando python(2) y Qt(4)

### Convertir pyQt.ui -> fileUi.py

- 	Instalar `pyqt4-dev-tools qt4-designer` 
-	`pyuic4 input.ui -o output.py`


### Launch
-	`python2 main.pyw`
